# Energy Management System (EMS) for Home Assistant

Умная система управления энергией для домов с солнечными панелями, аккумуляторами и динамическими тарифами.

## Возможности

- **7 Состояний Инвертора**: Гибкий конечный автомат (FSM) с приоритетами.
- **Power Guardian**: Контроль суммарной мощности дома для защиты от перегрузки сети.
- **Modularity**: Поддержка различных типов нагрузок (Бойлеры, Стиральные машины и др.).
- **Dynamic Dashboard**: Премиальный веб-интерфейс на базе Vue 3.
- **Occupancy Engine**: Учет количества людей для расчета резерва энергии.

## Установка

1. Добавьте URL этого репозитория в настройки вашего магазина аддонов Home Assistant.
2. Найдите "Energy Management System" и нажмите **Install**.
3. Настройте необходимые сенсоры в разделе **Configuration**.
4. Запустите аддон.

## Разработка

Проект построен на базе:
- **FastAPI** (Python)
- **Vue 3** (Frontend)
- **SQLite** (Persistence)
